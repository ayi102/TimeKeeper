Email settings

Use a Google App Password (Google Account → Security → 2-Step Verification → App passwords), not your normal password. Leave the password blank to keep the saved one.

Admin PIN

The PIN for this page and worker management. Default is 1234 — change it.

Backup

Sends a summary email with a full .db.gz database backup attached. The daily automatic send comes next; use this to test now.

Restore

Replaces ALL data on the tablet with a backup file (a .db.gz from a backup email, or a .db). Use it to load existing data, or to recover onto a new tablet. Best done from a computer or phone browser (the file must be on that device).

Kiosk

To leave the locked kiosk (to use the tablet or change settings), press and hold Back and Recent apps together to unpin — this works even if the app freezes. Re-open the TimeKeeper app to lock it back down.
